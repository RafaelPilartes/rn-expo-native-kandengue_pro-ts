# Roadmap — App Motorista (Kandengue Pro)

## Estado Actual

### ✅ Implementado

- Autenticação Firebase + verificação de status do motorista
- GPS tracking em foreground e background
- Gestão completa de corridas (aceitar/recusar/iniciar/terminar)
- Ride Summary modular (mapa + breakdown financeiro)
- Polyline trimming + rotação do ícone de posição
- RouteService com throttling (10s)
- Notificações push FCM (iOS + Android)
- Bottom Sheet fluido para gestão de corrida
- Histórico de corridas
- Suporte multilingue (PT + EN)
- Dark mode
- Awareness de rede offline
